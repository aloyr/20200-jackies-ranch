20200 Jackies Ranch Equipment Info

Existing Equipment

  • ADT Alarm

    All windows and doors are wired with sensors, and all main areas are wired with glass break sensors.

  • Honeywell Thermostat Controller

    This hooks up the 3 Honeywell thermostats to mobile phones and to home automation systems like SmartThings.

  • Jasco Smart Switches

    These switches can be controlled to home automation systems like SmartThings. They use the z-wave protocol.

  • Monoprice Whole Home Audio 10761

    This controls the audio system in the house. It can control up to 12 audio zones, and provide up to 6 distinct audio sources. There are 9 audio zones configured, each with its own control panel.

  • Protheus Pool Robot

    This robot cleans the pool on a preset schedule, and can be manually operated via app.

  • Rachio 2nd Generation Sprinkler

    This controls the sprinkler system and can be hooked up to a home automation system like SmartThings. It can control up to 16 zones. Current zones are front lawn, left side walkway, left side lawn, backyard living room, backyard master bedroom, backyard gate, right side lawn, backyard master shrubs, and pool trees.

  • Samsung Multipurpose Sensors

    These provide state information to a home automation system like SmartThings. When properly configured, the current sensors should report when entry ways and garage doors are opened and closed. In addition to that, these sensors will also report the temperature at each of their locations. These sensors use the zigbee protocol.

  • TP-Link T1600G-52TS Switch

    This is 1 out of 2 switches that connects all network ports in the house. A fiber loop connects both switches. Access its web interface with "admin" for both user and password.

  • TP-link TL-SG2452 Switch

    This is 1 out of 2 switches that connects all network ports in the house. A fiber loop connects both switches. Access its web interface with "admin" for both user and password.

  • Yale Real Living Locks

    These locks can operate stand-alone or it can be controlled by a home automation system like SmartThings. A working code will be sent via Realtor.

Suggested Equipment

  • Dakboard Dashboard CPU

    This is a simple option to easily deploy the Dakboard system that ran our family dashboard in the kitchen.

  • SmartThings Hub

    This is a simple-to-use home automation system that supports zwave, zigbee, and wi-fi protocols.
